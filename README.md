# Juego de ajedrez en Processing

Este repositorio incluye un sketch de Processing (`ChessGame.pde`) con un juego de ajedrez para 2 jugadores en local.

## Características

- Tablero 8x8 con piezas Unicode.
- Turnos alternados (blancas / negras).
- Reglas principales implementadas:
  - Movimiento legal por tipo de pieza.
  - Capturas.
  - Jaque, jaque mate y ahogado.
  - Enroque corto y largo.
  - Captura al paso.
  - Promoción automática a reina.

## Cómo ejecutarlo

1. Instala [Processing](https://processing.org/download).
2. Abre el archivo `ChessGame.pde` en Processing.
3. Presiona **Run**.

## Controles

- Clic en una pieza de tu color para seleccionarla.
- Clic en una casilla resaltada para mover.
- Clic en otra pieza propia para cambiar selección.
